# 基于Flutter的音乐播放器

由于刚写flutter，代码有点拉胯，有点丑 :sweat_smile: 

采用 Getx 状态管理

服务端：https://gitee.com/xujiangxing/kk_server


```
git clone https://gitee.com/xujiangxing/flutter_kuwo_music.git

flutter pub get

flutter run
```
